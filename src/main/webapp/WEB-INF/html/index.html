EV Rate Calculator


Instructions for Downloading Files

Create a folder on your Desktop by right-clicking and selecting "New" and "Folder". Name the folder "EV Rate Data"
Log in to customer's My Energy Account and click on "Green Button Download My Data"
Select the last button for "Export usage for a range of days" and change the start date to the previous year
Open the downloaded zip file
Select only the pge_ELECTRIC_interval_data file and click on extract files
Save the file in the newly created "EV Rate Data" folder under the Desktop tab by double clicking on "EV Rate Data" and then clicking export.
Go back to customer's My Energy Page and click on Green Button Download again. Select first button for "Export all bill totals"
Open the downloaded zip file and extract only pge_ELECTRIC_billing_data file
Extract this file to the same place as the first file, in the "EV Rate Data" folder by double clicking on "EV Rate Data" and then clicking export.
Go back to EV Calculator and click "Choose File" (or "Browse" if in Internet Explorer) for Interval Usage. Go to Desktop and then "EV Rate Data" and select correct file extension. Repeat for the Billing Periods file
Select Calculate Rates to see your results. Remember to delete EV Rate Data folder from Desktop when you are finished. Enjoy!
